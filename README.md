## 🛠️ Installation

1. Clone the repository:
```bash
git clone https://github.com/valleyworldz/xrpliquid.git
cd xrpliquid
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Configure your Hyperliquid credentials in `config/secure_creds.env`

4. Run the bot:
```bash
python run_bot.py
```
